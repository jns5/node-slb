# node-slb

A simple leaderboard using RocksDB for storage.

### Features(SO FAR):
- None

### Objectives:
- Highly configurable leaderboard
- Realtime update
- Filters
- Other data processing
- Expansible properties based in existing


### Technology Used:
- Backend:
    + REN stack (RocksDB + Express js + Node js)
    + Socket.io
- Frontend:
    + Plain old HTML5 + CSS3
    + Socket.io
    + Bootstrap
